Template literals:
Example:
var firstName = "John";
var lastName = "Doe";
var age = 30;
var city = "New York";

// Old way of concatenating strings
console.log("First Name: " + firstName + "last Name: " + lastName + "age: " + age + "city: " + city);
// New way of concatenating strings using template literals - introduced in ES6
console.log(`First Name: ${firstName} 
    last Name: ${lastName} 
    age: ${age} 
    city: ${city}`);

Array destructure:
Example:

Object destructure:
Example:
var person = {
    name: "John",
    city: "new york",
    age: 32
}

var { name, city } = person;

console.log(city);

Spread operator:
var fruits = ["apple", "cherry"];

console.log(copyArray);

Example:
var fruits = ["apple", "cherry"];
var veg = ["tomato", "orange"]
var copyArray = [...fruits, ...veg];

console.log(copyArray);
