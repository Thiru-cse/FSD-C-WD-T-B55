Position Properties:
need: even though if you have top, left, bottom and right
      which can be used to place/align the element on the page.
      1. Static
      2. Relative
      3. Absolute
      4. Fixed
      5. Sticky

1. Static Example:
    <div class="main">Static</div>
.main{
    position: static;
    right: 20px;
    height: 150px;
    width: 150px;
    background-color: #f0a500;
    color: #000;
    font-family: "PT Sans", sans-serif;
    text-align: center;
    border: 2px solid #000;
}

2. Relative Example:
    <div class="main">Static</div>
    .main{
    position: relative;
    right: 20px;
    left: 550px;
    top: 250px;
    height: 150px;
    width: 150px;
    background-color: #f0a500;
    color: #000;
    font-family: "PT Sans", sans-serif;
    text-align: center;
    border: 2px solid #000;
}

3. Absolute Example:
    <div class="parent">
        <div class="child">Absolute</div>
    </div>
.parent{
    position: relative;
    height: 300px;
    width: 300px;
    background-color: #f0a500;
    color: #000;
    font-family: "PT Sans", sans-serif;
    font-weight: 700;
    font-size: 30px;
    text-align: center;
    border: 2px solid #000;
    margin: 50px;
}

.child{
    background-color: salmon;
    position: absolute;
    top: 20px;
}

4. Fixed: the element is positioned relative to the browser window.

5. Sticky: the element positioned on the user's scroll position
           a sticky toggles between relative and fixed position when being scrolled.
