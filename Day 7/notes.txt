CSS - Cascading Style Sheet.
Why: To add customized to the website.

Inline CSS:
Example:
<h1 style="color: brown; font-size: 50px;">CSS Day 1</h1>
style: attribute
color: property
brown: value

Internal CSS:
Example:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Day 7 - CSS</title>
    <style>
        h1{
            color: brown;
        }
        p{
            color: blue;
        }
    </style>
</head>
<body>
<h1>CSS Day 1</h1>
<p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Magni, sed?</p>
<h1>welcome</h1>
</body>
</html>
