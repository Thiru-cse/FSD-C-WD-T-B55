if:
Syntax:
      if(condition){
        // Code to be executed if condition is true
      } else {
        // Code to be executed if condition is false
      }

Example:
var a = 2;

if (a % 2 == 0) {
  // Code to be executed if condition is true
  console.log("Even");
} else {
  // Code to be executed if condition is false
  console.log("Odd");
}

Switch Case:
switch(variable to be checked){
    case value1:
    code to be executed
    break;

    case value2:
    code to be executed
    break;

    default:
}

Example:
var category = "";

switch (category) {
    case "light":
    console.log("light is ON");
    break;

    case "Fan":
    console.log("Fan is ON")
    break;

    default:
        console.log("Everything is OFF");  
}
